Linux Shell

Dec 14, 2017

什么是Shell

  • Shell是文本解释器程序的统称。常见的Shell有sh、bash、ksh、rsh、csh等
  • Shell命令分为内建命令和外部可执行文件。
  • which date可以查看到命令对应的可执行文件的路径。
  • alias可以为命令创建别名。
  • type命令可以了解命令类型
  • echo $SHELL可以了解当前所使用的shell

命令行和参数

上述两条命令效果是相同的。
这种一个-的选项叫做短选项，引领的是一个字母
另一种两个--的选项叫做长选项，引领的是一个英文单词
选项可以携带参数，短选项使用空格链接。长选项使用等号链接
